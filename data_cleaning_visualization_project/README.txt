Data Cleaning & Visualization Mini Project

Files included:
- dataset.csv : Sample raw dataset with missing values and duplicates
- data_cleaning_visualization.py : Python script for cleaning and visualization

Steps to run:
1. Install required libraries:
   pip install pandas matplotlib seaborn

2. Run the python file:
   python data_cleaning_visualization.py

Outputs:
- Cleaned dataset
- Histogram plots
- Correlation heatmap
- Boxplot for outlier detection
